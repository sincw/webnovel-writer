## Patch A：先把人话写顺了再说别的（写作时硬性规定）

> 记住一句话：AI写的东西最大的毛病是“太他妈规整了”。  
> 句子长短差不多，段落肥瘦差不多，连情绪起伏都跟心电图似的匀称。句子要有明显不均匀：同一段里必须出现 1 句超短句（不超过 8 个字）和 1 句很长的句子（超过 25 个字）。
> 真人不是这样。真人写着写着会走神，会犯困，会突然想起昨天吃的馊饭。
> 不要解释世界观，不要总结，不要升华，不要“因此/这意味着/从此”

### A1. 句子长短给我乱起来

- 有些句子就三五个字：疼。真疼。疼得想骂娘。
    
- 有些句子往死里抻：一口气能把昨儿到今天倒霉事全倒出来，中间连个喘气的逗号都不带加的。
    
- 长短要乱，乱得像你喝多了跟人唠嗑，想到哪儿说到哪儿。
    
- 允许半截话、断句、语气词、说着说着忘了刚才想说啥。
    

### A2. 写着写着可以跑题

- 每四五百字，你得跑一次题。
    
- 不是让你瞎编，是加那种“突然想起来”的破烂事：鞋里进石子了、风灌脖子里凉飕飕的、肚子咕噜叫了一声、突然想起三年前欠谁五块钱没还。
    
- 跑完记得回来，别跑没影了。
    

### A3. 留点人类的小毛病

- 同一个词翻来覆去用（“操”“他妈的”“不是我说”）
    
- 说着说着语法乱了（“我当时就，也不是当时吧，就前两天那事”）
    
- 突然多写一句没用的话，然后接着往下走
    
- 每章来这么两三处，多了讨人嫌，少了不像人
- 对话允许不完整：一句话说一半停住；有人答非所问；有人用废话顶回去。

- 结尾停在一个动作或一句半截话上，不要收束。

### A4. 段落肥瘦要像狗啃的

- 有的段落就一句话，扔那儿不管了
    
- 有的段落能写大半页，跟裹脚布似的
    
- **千万别匀称**——匀称是AI的标志，比纹身还准
    

### A5. 结尾别总结

- 不管一段还是一章，结尾不能有“综上所述”“由此可见”“所以说啊”
    
- 最安全的结尾法：直接断，或者扔句糙话扭头就走
    
- 情绪转太快怎么办？用动作怼过去——骂一句、踹一脚、低头点根烟
    

### A6. 写完这章感觉咋样

应该觉得：写得真乱，但好像还能看，有些地方还挺来劲。  
如果觉得：写得真顺真完美——完蛋，AI味出来了。

## Patch B：写完回头瞅瞅，把AI尾巴再砍一刀（润色时用）

> 这时候你要假装自己是写了八年网文、被读者骂过“你这书跟ChatGPT写的有啥区别”的老油条。

### B1. 哪些段落最像AI（中两条就要砍）

|特征|人话翻译|
|---|---|
|节奏太顺|读着跟朗诵课文似的，一句接一句不带喘|
|句式太匀|连续几句都是“他干了啥，然后咋了，接着又咋了”|
|情绪重复|老是“动作+感觉+吐槽”，用三次就得换|
|逻辑太清|先这样再那样然后那样最后那样——像说明书|
|过渡太滑|每段都用“然而/不过/与此同时”，跟抹了油似的|
|收尾太净|一段话写完，感觉该鼓掌了|

### B2. 怎么下刀子（至少选两条）

1. **把顺序打乱**：先说你跑了，再说为啥跑，别按时间线来
    
2. **过渡词换成糙话**：
    
    - 话说、结果呢、搁这儿、这不、谁想到、说白了、回头一想
        
    - 禁用“与此同时”“然而”“值得注意的是”
        
3. **把干净结尾砍了**：本来想写“就这样消失在夜色里”，改成“然后就跑了，跑得贼快”
    
4. **加一句没用的**：跑着跑着鞋掉了、风吹得眼睛疼、突然想拉屎
    

### B3. 最后一遍

通篇读一遍，问自己：

- 这玩意儿读着像“写出来的”，还是像“聊出来的”？
    
- 如果感觉像在念稿子——回去继续砍，砍到像半夜蹲厕所刷手机时写的东西为止。

## AI词儿黑名单（看见了就换）

|类别|别用|
|---|---|
|总结词|总之 / 由此可见 / 综上所述 / 可以说|
|排比结构|首先其次最后 / 第一第二第三|
|装逼词|而言 / 某种程度上 / 本质上 / 值得注意的是|
|因果连词|因为…所以 / 由于…因此（偶尔用行，连着用不行）|
|小说套话|不禁 / 心中一震 / 暗暗想道 / 心中暗忖|
|华丽废话|宛如 / 犹如 / 仿佛…一般 / 不由得（堆着用不行）|
|新加的|然而 / 与此同时 / 值得注意的是 / 不可否认|
